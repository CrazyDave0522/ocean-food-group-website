## ADDED Requirements

### Requirement: Header Navigation
The site SHALL include a persistent header with links to the main pages: Home, Media Reviews, Franchise, Careers, Contact. The header SHALL be present on all public pages and clearly visible on desktop and mobile.

#### Scenario: Header presence
- **WHEN** a user visits any public page
- **THEN** the header is visible and contains links to Home, Media Reviews, Franchise, Careers, Contact

### Requirement: Footer Navigation
The site SHALL include a footer containing links to `Terms & Conditions` and `Privacy Policy` that navigate to `/terms` and `/privacy` respectively.

#### Scenario: Footer presence
- **WHEN** a user scrolls to the bottom of any public page
- **THEN** the footer is visible and contains links to Terms & Conditions and Privacy Policy
